## 1. 공통 안내

### 1.1 접속 방법

배포된 웹사이트 URL로 접속하면 로그인 화면이 표시됩니다.

### 1.2 로그인 방식

| 역할 | 로그인 방법 |
|------|-------------|
| **학생** | 학번 + 이름 입력 (안전교육 이수자 명단에 등록되어 있어야 함) |
| **근로학생** | 관리자가 발급한 아이디 + 비밀번호 입력 |
| **관리자** | 관리자 아이디 + 비밀번호 입력 |

- "로그인 정보 기억하기" 체크 시 다음 접속 때 자동으로 입력됩니다.
- 블랙리스트에 등록된 학생은 로그인이 제한됩니다.

### 1.3 로그인 화면 부가 기능

로그인 전에도 아래 기능을 이용할 수 있습니다:

- **안전교육 수료증 업로드**: 학번, 이름, 학년, 전공(5년제/4년제), 이메일, 수료증 파일을 제출
- **일반 문의 접수**: 제목, 내용, 이름, 연락처를 입력하여 문의 (익명 가능)
- **학사 공지 확인**: 국민대학교 학사공지, 장학공지, 사회봉사 공지 열람
- **커뮤니티 게시판**: 익명 게시글 작성 및 댓글 참여
- **전시회 정보**: 건축대학 전시회 일정 및 포스터 확인

## 2. 학생 포털

### 2.1 첫 이용 안내

1. 안전교육을 이수합니다.
2. 로그인 화면에서 **안전교육 수료증 업로드**를 통해 수료증을 제출합니다.
3. 관리자가 수료증을 승인하면 이메일로 알림이 옵니다.
4. 안전교육 이수자 명단에 등록된 후 **학번 + 이름**으로 로그인합니다.

> 안전교육 미이수 상태에서는 실기실 예약과 물품 대여가 제한됩니다. 출력 신청, 이용내역, 문의는 이용 가능합니다.

---

### 2.2 대시보드

로그인 후 첫 화면입니다. 전체 현황을 한눈에 확인할 수 있습니다.

| 패널 | 내용 |
|------|------|
| **실기실 예약** | 예정된 예약 건수 및 목록 (최대 3건 표시) |
| **물품 대여** | 현재 대여 중인 물품 현황 |
| **출력 신청** | 진행 중인 출력 요청 현황 |
| **내 이용내역** | 최근 예약/대여 이력 요약 |
| **문의 내역** | 접수한 문의 현황 (대기/답변완료) |

각 패널의 버튼을 누르면 해당 탭으로 이동합니다.

---

### 2.3 실기실 예약

**예약 절차:**

1. 좌측에서 **실기실**을 선택합니다.
2. **날짜**를 선택합니다 (내일 이후 날짜만 선택 가능).
3. 시간표에서 원하는 **시간대**를 클릭합니다.
   - 초록색: 예약 가능
   - 회색: 이미 예약된 시간
4. **목적**과 **인원**을 입력합니다.
5. **예약 신청** 버튼을 클릭합니다.

**예약 후:**
- 예약 확정 이메일이 본인 이메일로 발송됩니다.
- 대시보드 및 내 이용내역에서 예약 현황을 확인할 수 있습니다.

**예약 취소:**
- 내 이용내역 탭에서 해당 예약의 **취소** 버튼을 클릭합니다.
- 취소 확인 이메일이 발송됩니다.

---

### 2.4 물품 대여

**대여 절차:**

1. 상단 카테고리 필터(전체/가공장비/수공구/전자제품/기타)로 물품을 탐색합니다.
2. 원하는 **물품 하나**를 선택합니다 (한 번에 1개씩 신청).
   - "품절"인 물품은 선택할 수 없습니다.
3. 우측에서 **반납 예정일**을 설정합니다 (물품별 최대 대여일 이내).
4. 필요시 **비고**를 입력합니다.
5. **대여 신청** 버튼을 클릭합니다.

**대여 후 흐름:**

| 단계 | 상태 | 설명 |
|------|------|------|
| 1 | 수령 대기 | 신청 완료, 근로학생이 물품 준비 중 |
| 2 | 준비 완료 | 근로학생이 준비를 마침 → 이메일 알림 수신 |
| 3 | 반납 | 교학팀에 반납 후 근로학생이 반납 처리 |

- 여러 물품을 빌리려면 하나 신청 후 다른 물품을 추가로 신청합니다.
- 보증금이 필요한 물품은 수령 시 보증금을 납부합니다.

---

### 2.5 출력 신청

**신청 절차:**

1. **용지 크기**를 선택합니다 (A4, A3, A2, A1, A0).
2. **색상 모드**를 선택합니다 (흑백/컬러).
3. **매수**를 입력합니다.
4. 금액을 확인하고 표시된 **카카오뱅크 계좌**로 송금합니다.
5. **출력할 파일**을 업로드합니다.
6. **송금 완료 캡처**를 업로드합니다.
7. 필요시 비고를 입력하고, 긴급 수령이 필요하면 체크합니다.
8. **출력 신청하기** 버튼을 클릭합니다.

**신청 후:**
- 접수 확인 이메일이 본인 이메일로 발송됩니다.
- 근로학생이 출력을 완료하면 **출력 완료 이메일**이 발송됩니다.
- 건축대학 출력실(본관 3층)에서 수령합니다.
- 운영시간: 평일 10:00~17:00 (점심시간 12:00~13:00 제외)

**출력 가격표:**

| 용지 | 흑백 | 컬러 |
|------|------|------|
| A4 | 50원 | 200원 |
| A3 | 100원 | 500원 |
| A2 | 500원 | 2,000원 |
| A1 | 1,000원 | 4,000원 |
| A0 | 2,000원 | 8,000원 |

---

### 2.6 내 이용내역

- 실기실 예약과 물품 대여 내역을 통합하여 확인합니다.
- 예약 상태: 승인 / 수령대기 / 수령완료 / 취소 / 반려 / 반납
- 승인 전 또는 수령 대기 중인 예약은 **취소**할 수 있습니다.

### 2.7 문의 내역

- 본인이 접수한 문의 목록과 답변 내용을 확인합니다.
- 대기중 / 답변완료 상태로 분류됩니다.

## 3. 근로학생 포털

### 3.1 개요

근로학생은 학생들의 예약, 물품 대여, 출력 요청을 처리하고 문의에 답변하는 역할입니다.

---

### 3.2 대시보드

로그인 후 첫 화면으로, 처리가 필요한 항목을 한눈에 볼 수 있습니다.

- **알림**: 새로운 요청이 들어오면 알림 배지가 표시됩니다. 개별 확인 또는 전체 읽음 처리가 가능합니다.
- **실기실 예약 현황**: 대기 중인 예약을 승인하거나 반려할 수 있습니다.
- **물품 대여 현황**: 준비가 필요한 대여 요청 목록을 확인합니다.
- **출력 요청 현황**: 대기 중인 출력 건수를 확인합니다.
- **방문자 통계**: 누적 방문 수를 표시합니다.

---

### 3.3 물품 대여 관리 (대시보드 내)

**처리 절차:**

| 학생 신청 상태 | 근로학생 액션 | 결과 |
|---------------|-------------|------|
| 수령 대기 (준비 필요) | **준비 완료** 버튼 클릭 | 학생에게 준비 완료 이메일 발송 |
| 준비 완료 (학생 수령 후) | **반납 처리** 버튼 클릭 | 반납 완료 처리 |

- 준비 완료 시 반납 체크리스트가 생성됩니다.
- 반납 시 체크리스트 항목을 모두 확인한 후 반납 처리합니다.

---

### 3.4 출력 관리

상단 필터로 상태별 요청을 분류할 수 있습니다:

| 필터 | 내용 |
|------|------|
| **대기** | 새로 접수된 출력 요청 |
| **출력중** | 현재 출력 진행 중인 요청 |
| **완료** | 출력 완료 및 취소된 요청 |
| **전체** | 모든 요청 |

**처리 절차:**

1. 대기 중인 요청에서 학생 정보, 파일, 송금 캡처를 확인합니다.
2. 출력 시작 시 **출력 시작** 버튼을 클릭합니다 → 상태가 "출력중"으로 변경됩니다.
3. 출력 완료 후 **출력 완료** 버튼을 클릭합니다 → 학생에게 **출력 완료 이메일**이 발송됩니다.
4. 문제가 있으면 **취소** 처리할 수 있습니다.

- 긴급 수령 요청(🚨)은 우선적으로 처리합니다.

---

### 3.5 문의 관리

- **대기중** / **답변완료** 필터로 문의를 분류합니다.
- 문의 내용과 학생 연락처를 확인합니다.
- **답변 작성**: 텍스트로 답변을 입력하여 등록합니다.
- **직접 연락 완료**: 전화/대면 등으로 처리한 경우 완료 처리합니다.
- 부적절한 문의는 **삭제**할 수 있습니다.

---

### 3.6 일지 (로그)

- 시스템에서 자동 기록된 모든 활동 이력을 확인합니다.
- 유형별 필터: 예약 / 출력 / 물품 / 문의 / 관리자 / 일반
- 검색 기능으로 특정 이력을 찾을 수 있습니다.
- 조회 전용이며 수정/삭제는 불가합니다.

## 4. 관리자 포털

### 4.1 개요

관리자는 시스템 전반을 관리하며, 근로학생 계정 관리, 경고/블랙리스트 관리, 물품 관리, 수료증 승인 등을 담당합니다.

---

### 4.2 근로학생 계정

**계정 생성:**

1. **새 계정 추가** 버튼을 클릭합니다.
2. 이름, 근무시간, 아이디(3자 이상), 비밀번호(4자 이상)를 입력합니다.
3. **저장** 버튼을 클릭합니다.

- 아이디는 중복 불가입니다.
- 기존 계정의 정보를 **수정**하거나 **삭제**할 수 있습니다.
- 비밀번호는 눈 모양 아이콘으로 표시/숨김 전환이 가능합니다.

---

### 4.3 경고 / 블랙리스트

**경고 누적:**

1. 학번, 이름, 사유를 입력합니다.
2. **경고 추가** 버튼을 클릭합니다.
3. 동일 학생에게 경고를 추가하면 횟수가 누적됩니다.

- 학생 포털에서 경고 횟수가 배지로 표시됩니다.

**블랙리스트:**

1. 학번, 이름, 사유를 입력합니다.
2. **블랙리스트 추가** 버튼을 클릭합니다.

- 블랙리스트에 등록된 학생은 **로그인이 차단**됩니다.
- 해제하려면 해당 항목을 **삭제**합니다.

---

### 4.4 물품 관리

실기실에서 대여 가능한 물품 목록을 관리합니다.

**물품 추가:**

1. **새 물품 추가** 버튼을 클릭합니다.
2. 아래 항목을 입력합니다:
   - 물품명
   - 카테고리 (가공장비 / 수공구 / 전자제품 / 기타)
   - 아이콘 (이모지)
   - 총 수량, 가용 수량
   - 최대 대여일
   - 보증금 필요 여부
3. **저장** 버튼을 클릭합니다.

**물품 수정/삭제:**

- 각 물품 카드의 **수정** 버튼으로 정보를 변경합니다.
- **삭제** 버튼 클릭 후 확인하면 물품이 제거됩니다.

> 학생이 물품을 대여하면 가용 수량이 자동으로 1 감소합니다.

---

### 4.5 커뮤니티 / 전시

**전시회 관리:**

- 전시 제목, 장소, 기간, 인스타그램 URL, 설명, 포스터 이미지를 등록합니다.
- 등록된 전시 정보는 로그인 화면에서 모든 사용자가 확인할 수 있습니다.
- 수정 및 삭제가 가능합니다.

**커뮤니티 관리:**

- 익명 게시판의 모든 게시글과 댓글을 확인합니다.
- 부적절한 게시글 또는 댓글을 개별 **삭제**할 수 있습니다.

---

### 4.6 수료증 관리

학생이 제출한 안전교육 수료증을 심사합니다.

**처리 절차:**

1. 대기 중인 수료증 목록에서 학생 정보와 파일을 확인합니다.
2. 수료증 이미지를 클릭하여 원본을 확인합니다.
3. 정상이면 **승인** 버튼을 클릭합니다.
   - 학생에게 승인 이메일이 발송됩니다.
   - 해당 학생의 안전교육 이수 상태가 활성화됩니다.
4. 문제가 있으면 **반려** 버튼을 클릭합니다.
   - 학생이 재제출해야 합니다.

---

### 4.7 관리 이력

- 관리자의 모든 작업 이력을 확인합니다.
  - 계정 생성/수정/삭제
  - 경고/블랙리스트 변경
  - 수료증 승인/반려
  - 물품 추가/수정/삭제
  - 커뮤니티 관리
  - 전시회 관리
- 유형별 필터와 검색 기능을 지원합니다.

---

### 4.8 연동 설정

구글 시트 연동 URL을 설정합니다.

- **예약 Webhook URL**: 실기실 예약 데이터를 구글 시트에 동기화
- **출력 Webhook URL**: 출력 신청 데이터를 구글 시트에 동기화
- URL 형식: `https://script.google.com/macros/s/XXX/exec`
- 저장 후 예약/출력 신청 시 자동으로 구글 시트에 데이터가 기록됩니다.

## 자주 묻는 질문 (FAQ)

**Q. 로그인이 안 됩니다.**
> 안전교육 이수자 명단에 등록되어 있는지 확인해주세요. 수료증을 먼저 제출하고 관리자 승인을 받아야 합니다.

**Q. "안전교육 미이수"라고 뜹니다.**
> 안전교육 수료증을 업로드하고 관리자 승인을 기다려주세요. 승인 후 재로그인하면 해소됩니다.
